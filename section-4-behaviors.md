# Section 4 — Culture Behaviors (1,000 chars)

## Required Behaviors

Three behaviors are required by the form:
1. Grow with Curiosity and Adaptability
2. Collaborate and Include Others to Build Better Outcomes
3. Be Bold, Move Fast, with Purpose

## Optional Additions (from prior Connect)

If your past review highlighted additional behaviors, add them here. Common examples:
4. Systematic Thinking Over Heroics
5. Visibility and Proof of Work

## Format

```
[Behavior label]: [One sentence on approach]. [One specific example from the logs].
```

## Rules

- ~180-200 chars per behavior if using 5 behaviors (stay within 1,000 total)
- Every behavior grounded in a specific example — no abstract claims
- Approach sentence first, example second
- Avoid: "I demonstrated [behavior] by..." — embed the behavior, don't announce it
